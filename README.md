## ‚ûï Adding a New Image (Step-by-Step, No Collaborator Access Required)

You **do not need to be a collaborator** on this repository to add images. Images are contributed using a **Pull Request (PR)**.

### Exact steps to add an image:

1. **Fork the repository**

   * Click the **Fork** button in the top-right corner of this page
   * GitHub will create a copy of the repository under your own account

2. **Open your forked repository**

   * You will be redirected to: `your-username/public-images`

3. **Upload the image**

   * Click **Add file ‚Üí Upload files**
   * Drag and drop your image file onto the page
   * (Optional) Place the image into a relevant folder (for example: `memes/` or `slack/`)

4. **Commit the image**

   * Scroll to the bottom of the page
   * Enter a short commit message (for example: `Add shared channel image`)
   * Click **Commit changes**

5. **Create a Pull Request**

   * After committing, GitHub will show a banner at the top of the page
   * Click **Compare & pull request**
   * Review the changes
   * Click **Create pull request**

6. **Done üéâ**

   * The image will be reviewed and merged
   * No further action is required from you
